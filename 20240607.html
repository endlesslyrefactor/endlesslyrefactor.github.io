Tonight I've decided to do some dev work on my laptop which has not worked on this repo before. Immediately I hit some problems

Firstly I needed to install a newer version of node via nvm. The download seemed to work but npm install threw an error about npm-cli

I think that the installer was failing to update the path variable for node because a home automation script was running in node at the same time

In the end the fix was quite simple. I updated nvm and rebooted my machine to be safe. And then uninstalled and reinstalled the version of node I wanted to use

That let me install packaged but running the server threw errors because main.js couldn't be found. I realised I had installed TypeScript globally on the other machine which didn't seem like the best idea. I added it to the packages.json so it would be installed automatically from now on 

      
        npm install typescript --save-dev

Then I remembered that I was missing something obvious from the readme. I added a section about running npx tsc to build the TypeScript

After all that I feel like I should just do something simple tonight. I will try and extend a todo so it is a description

As a description is just another string backed value object I created a TodoDescription based off the TodoName code I already had. But I increased the maximum allowable size to 500. That number was picked from thin air but it should be very easy to chain later

I then added the description to the Todo class and modified the tests and the main file to know about them

I think I will call it there tonight. Next I should make the description editable and make a UI to set it
